# Portafolio Profesional - Contexto de Agente

## Descripción General
Este es un proyecto de portafolio profesional construido con **Vite** y **React**. El objetivo es crear una experiencia web premium, modular y optimizada.

## Stack Tecnológico
- **Core:** React 19+
- **Bundler:** Vite
- **Estilos:** Vanilla CSS con variables HSL
- **Estado:** React Hooks
- **Estructura:** Modular (Componentes > Secciones)

## Convenciones de Código
- **Componentes:** Usar Functional Components con PascalCase.
- **Estilos:** Priorizar el sistema de diseño en `src/styles/variables.css`.
- **Rutas:** Mantener lógica de negocio en `src/hooks` y constantes en `src/constants`.
- **Imágenes:** Usar la carpeta `src/assets`.

## Comandos Útiles
- `npm run dev`: Iniciar servidor de desarrollo.
- `npm run build`: Generar build de producción.
- `npm run preview`: Previsualizar build localmente.

## Notas para el Agente
- Priorizar **Aesthetics** y micro-animaciones.
- Mantener el código limpio y libre de redundancias.
- Usar nombres descriptivos en español/inglés según el contexto del usuario.
